# Vue3 Webpack Boilerplate

A Vue 3 simple boilerplate with Webpack 5 without Vue-cli or Vite.

Just practise.

Do not use in production.

[中文说明](README_CN.md)

## Includes

- Webpack 5
- Vue 3
- Vue Router 4
- pinia 2
- Babel 7
- TypeScript 4
- postcss 8
- sass 1.5
- dotenv 16

## Setup

Recommond use pnpm.

```shell
pnpm install
```

## Start

Start server with hot-reloads for development.

```
pnpm start
```

## Build

Build project for production.

```
pnpm build
```
